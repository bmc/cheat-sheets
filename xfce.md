# Emacs key bindings

    xfconf-query -c xsettings -p /Gtk/KeyThemeName -s Emacs

# Fuzzy antialiasing

Select *Settings > Appearance* and, under the *Fonts* tab, play with the
*Sub-pixel order* setting.
